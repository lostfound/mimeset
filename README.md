mimeset
=======

mimeset script makes application the default application for opening supported files

## Installation

Type this command as root:

    $ make install"
or

    $ PREFIX=/usr make install

## Uninstallation

    $ make uninstall
or
    $ PREFIX=/usr make uninstall

## Usage

    $ mimeset appname

    $ mimeset --listapps

    $ mimeset --mimes appname
